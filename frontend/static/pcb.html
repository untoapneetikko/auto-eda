Gerber Files
RS-274X Gerber + Excellon drill file. Standard format accepted by JLCPCB, PCBWay, OSHPark, and most manufacturers. Includes copper, silkscreen, solder mask and board outline.
🗂
KiCad PCB
KiCad .kicad_pcb format (v20221018). Import into KiCad 7+ for advanced editing, 3D view, or additional DRC. Components and traces are included.
📐
DXF
DXF format for importing board outlines and drawings into mechanical CAD tools. Choose which layer to export from the Tools tab.
⬇
PCB JSON
Native PCB JSON format. Download and reload the board later using the Load JSON button on the Boards tab, or share with others using this tool.
