# Summary: Group/Ungroup Feature — Schematic GUI

**Date:** 2026-03-13

## Changes Made

### `frontend/static/js/index-schematic-editor.js`

1. **Data model** — `newProject()` now initialises `project.groups = []`; `loadProject()` migrates old projects with `if (!this.project.groups) this.project.groups = []`.

2. **Undo/redo** — `_saveHist()` serialises `groups` into the history snapshot (key `g`); both `_undo()` and `_redo()` restore `project.groups = s.g || []`.

3. **Context menu** — `contextmenu` event on the SVG now calls `_showContextMenu(e)` instead of `_cancel()`. Full context menu implementation added with `_showContextMenu()` / `_hideContextMenu()`.

4. **`_cancel()`** — now calls `_hideContextMenu()` before switching tool.

5. **Keyboard shortcuts** — `_key()` handles `Ctrl+G` → `groupSelected()` and `Ctrl+Shift+G` → `ungroupSelected()`.

6. **Named group auto-select in `_down()`** — clicking any member of a named group auto-selects all group members and starts a group drag immediately.

7. **`_delSelected()` group cleanup** — after deleting elements, stale group member IDs are pruned; empty groups are removed.

8. **`_render()`** — renders group bounding boxes via `_groupH()` before wires, and DRC unconnected-pin markers via `_drcH()` after junctions.

9. **`destroy()`** — calls `_hideContextMenu()` to clean up on teardown.

10. **New methods added:**
    - `_getGroupByMember(id)` — find group containing an element ID
    - `_groupBBox(group)` — compute bounding box over all group members
    - `_groupH(group)` — render dashed SVG rect + name label
    - `_drcH()` — render orange circles on unconnected component pins
    - `groupSelected()` — create a named group from multiSelected
    - `ungroupSelected()` — dissolve groups touching the selection
    - `_renameGroup(grp)` — prompt-based rename
    - `_showFlash(msg)` — flash a message in the status-hist element
    - `_showContextMenu(e)` — right-click context menu
    - `_hideContextMenu()` — dismiss and remove the context menu DOM node

### `frontend/static/index.html`

1. **CSS** — `.sch-ctx-menu`, `.sch-ctx-item`, `.sch-ctx-item.danger`, `.sch-ctx-kbd`, `.sch-ctx-sep` added after `.toolbar-spacer`.

2. **Toolbar buttons** — "Group" (Ctrl+G) and "Ungroup" (Ctrl+Shift+G) buttons added before the SVG export separator.
