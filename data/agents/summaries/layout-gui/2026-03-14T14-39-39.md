# Group-rotate all selected components with R/G

## What changed
- Added `rotateSelGroup(deg)` method to `PCBEditor`:
  - **Single selection**: rotates the component in place (same as before)
  - **Multi-selection**: computes the centroid of all selected component positions, rotates each component's position around that centroid by 90° CW, and increments each component's individual rotation by 90° — so the whole group pivots together
- `R` key: now calls `rotateSelGroup` (previously only rotated `selectedComp`, ignoring multi-selection)
- `G` key: same fix
- `rotSel()` button (↻ in the info panel): now delegates to `rotateSelGroup`
