# PMA3-83LNW+ example application circuit built

## What changed
- Built complete Fig. 1 application circuit from the Mini-Circuits datasheet
- RF path: RF_IN → C2 (10pF series coupling) → pad 2 → U1 (PMA3-83LNW+) → pad 8 → C3 (100pF DC block) → RF_OUT
- Bias path: VDD (5V/6V) → L2 (39nH RF choke) → pad 8 (RF-OUT/DC-IN)
- Input shunt: pad 2 → L1 (18nH) → GND (DC bias return + ESD path)
- Supply bypasses: C1 (0.01µF, HF) and C4 (100nF, LF) from VDD to GND
- GND paddle included; pads 11 & 12 left floating (per datasheet exception)
- Pads 1, 3–7, 9–10 all tied to GND (per test board practice)

## Files
- `data/outputs/example_schematic.json` — KiCad-schema-compatible schematic with 7 components, 6 nets
- `frontend/static/library/PMA3_83LNW/profile.json` — `example_circuit` updated via PUT API with full visual schematic (12 components, 11 wires, 3 net labels)

## Notes
- L2 and C3 are mandatory — missing either will damage the amplifier or downstream components
- VDD net rail at y=160 feeds C1, C4 (bypass) and L2 (bias choke) from a single horizontal rail
- Component BOM matches datasheet p.4 exactly: L1=18nH, L2=39nH, C1=0.01µF, C2=10pF, C3=100pF
